## [boostcourse] 2. HTML Layout 태그

### 1. HTML tag의 종류

|**태그**|**기능**|
|---|-------|
|a href|링크|
|img src|이미지|
|ul/li|목록|
|h|제목 h1,h2|
|p|문단, 텍스트 집합을 표현|
|div|블럭태그|


### 2. HTML Layout 태그

|**태그**|**기능**|
|---|-------|
|header|머리글(제목,헤더)
|nav|네비게이션 메뉴, 다른 문서로의 하이퍼링크
|aside|좌.우측 사이드 공간
|footer|바닥글
